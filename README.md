# SolarExplorer-app-javafx
JavaFX educational quiz game for children using MVC architecture and the Factory Design Pattern

This app was designed as an educational solar system quiz game for primary school 
children. A solar System is drawn on the screen with the sun at the centre and coloured 
planets orbiting the sun – each planet extends the provided Game Object class. Each 
round, a mystery planet is highlighted with a glowing white ring and the player must try 
to identify it. Three types of questions are used – guess the planet from clues, true or 
false facts and guess a planet's statistics, such as their temperature or the number of 
moons they have. A hint system reveals a clue at a time, deducting 50 points per use 
and rewarding players who need fewer hints. The game was built using Eclipse, Maven 
and the javafx-architecture-simple as demonstrated in the module workshops (Drumm, 
2026). Two design patterns were implemented - MVC and Factory.

## Features

- 🌍 Interactive Solar System visualisation
- 🪐 Orbiting planets surrounding the Sun
- ❓ Multiple educational quiz types
- 💡 Progressive hint system
- ⭐ Live score tracking
- 📊 Hint usage tracking
- 🎯 Mystery planet highlighting
- 🔄 Random question generation
- 🎮 Child-friendly graphical interface
